## 一、监控指标概述

### 1.1 指标分类

| 分类 | 前缀 | 说明 |
|------|------|------|
| 规则引擎 | `fly2_rule_*` | 规则执行相关指标 |
| 趋势分析 | `fly3_trend_*` | 趋势分析和预测指标 |
| 工具中心 | `fly5_tool_*` | 工具注册和执行指标 |
| 消息队列 | `fly5_mq_*` | 消息队列相关指标 |
| 系统级 | `system_*` | 基础设施指标 |

### 1.2 指标属性

每个指标包含以下属性：
- **名称**: 指标唯一标识
- **类型**: gauge (当前值) / counter (累积值) / histogram (分布)
- **单位**: 指标单位
- **描述**: 指标含义

## 七、关键性能指标 (KPI)

### 7.1 SLA 目标

| 指标 | 目标值 | 告警阈值 |
|------|--------|----------|
| 规则执行成功率 | ≥99.9% | <99.5% |
| 预测准确度 | ≥85% | <75% |
| 消息队列延迟 | <100ms | >500ms |
| API 响应时间 P99 | <200ms | >500ms |
| 系统可用性 | ≥99.95% | <99.9% |

### 7.2 Dashboard 看板

建议创建以下 Dashboard：
1. **系统概览**: 整体健康状态、关键指标
2. **规则引擎**: 执行统计、缓存命中率
3. **预测引擎**: 预测趋势、准确度监控
4. **消息队列**: 吞吐量、延迟、死信监控
5. **资源池**: 利用率、等待时间
